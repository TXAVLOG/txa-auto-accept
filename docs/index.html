Phương thức cài đặt

Chọn 1 trong 2 cách sau tùy theo editor của bạn.

Tải file VSIX (Khuyên dùng)
Cách cài (VS Code): Tải file .vsix về máy. Mở VS Code -> tab Extensions (Ctrl+Shift+X) -> chọn dấu chấm ... -> bấm Install from VSIX và chọn file vừa tải.
Cài đặt qua Open VSX
Cách cài: Extensions đã có mặt chính thức trên nền tảng mã nguồn mở Open VSX Registry. Bấm vào nút trên để chuyển hướng tới trang chợ extension.
